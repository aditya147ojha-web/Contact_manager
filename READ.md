PROJECT DESCRIPTION

This is a Python-based contact management system that allows users to store, manage, and organize contacts using dictionaries. It supports full CRUD operations (Create, Read, Update, Delete) along with data validation and file storage using JSON.

---

FEATURES

* Add new contacts
* Search contacts (partial matching supported)
* Update existing contact details
* Delete contacts
* View all contacts
* Save and load contacts using JSON file
* Export contacts to CSV file
* Display basic statistics
* Input validation and error handling

---

HOW TO RUN

1. Open terminal or command prompt

2. Navigate to project folder:
   cd week3-contact-manager

3. Run the program:
   python contacts_manager.py

---

DATA STRUCTURE

Contacts are stored in dictionary format:

Name → Details

Example:

John Doe
Phone: 1234567890
Email: [john@example.com](mailto:john@example.com)
Group: Friends
