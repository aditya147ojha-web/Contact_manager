Project: Week 3 – Functions & Dictionaries

---

PROJECT DESCRIPTION

A Python-based contact management system that allows users to manage contacts using dictionaries. It supports adding, searching, updating, deleting contacts, and saving data using JSON.

---

FEATURES

* Add new contacts
* Search contacts (partial match)
* Update contact details
* Delete contacts
* View all contacts
* Save and load contacts (JSON)
* Export contacts to CSV
* Display statistics
* Input validation and error handling

---

HOW TO RUN

1. Open terminal

2. Navigate to project folder:
   cd week3-contact-manager

3. Run program:
   python contacts_manager.py

4. Run test file:
   python test_contacts.py

---

DATA STRUCTURE

Contacts are stored in dictionary format:

contacts = {
"John Doe": {
"phone": "1234567890",
"email": "[john@example.com](mailto:john@example.com)",
"group": "Friends"
}
}

---

CONCLUSION

This project demonstrates the use of functions, dictionaries, file handling, and validation to build a real-world application.
